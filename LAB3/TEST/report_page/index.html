  • Appendix

    Some Clarification

    * Summarized Score are calculated by:

    Score = SleepQuality * SleepTimeRatio * ActivityTimeRatio * (MoodPositive / MoodNegative)

    SleepTimeRatio = SleepTime / DayTime;

    ActivityTimeRatio = ActivityTime / DayTime;

    ** Again, this is an ASSUMPTION equation, which WILL BE REPLACED in the future for improving accuracy
